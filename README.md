# Election Data Analysis: Chandigarh & Ladakh (1967 - 2024)

## Project Overview
This project contains a comprehensive dataset of Lok Sabha election results for the constituencies of **Chandigarh** and **Ladakh**. It tracks electoral trends, winning margins, and candidate demographics over nearly six decades.

As a final-year Computer Science student, I developed this dataset to analyze historical voting patterns and understand the impact of narrow victory margins in Indian elections.

## Dataset Description
The repository includes CSV files with the following key columns:
* **YEAR**: The election year.
* **WINNER CANDIDATE NAME**: Name of the successful candidate.
* **PARTY NAME**: Political party affiliation.
* **CATEGORY**: Official constituency classification (General/SC/ST). Note: Both Chandigarh and Ladakh are **General** seats.
* **MARGIN (VOTES)**: The difference in votes between the winner and the runner-up.
* **VOTE %**: The percentage of voter turnout for that election year.

## Key Insights
* **Narrowest Margin**: The 2014 Ladakh election was decided by a historic margin of only **36 votes**.
* **Uncontested Wins**: The 1967 Ladakh election saw an uncontested victory for Kushok Bakula.
* **Consistency**: Tracking candidates like Pawan Kumar Bansal (INC) and Kirron Kher (BJP) highlights long-term political shifts in Chandigarh.

## How to Use
The data is provided in `.csv` format, making it easy to import into:
1.  **Python (Pandas)** for data cleaning and analysis.
2.  **Power BI / Tableau** for creating election dashboards.
3.  **Excel/Google Sheets** for quick reference and filtering.

## Technologies Used
* Data Collection: Manual curation from ECI records.
* Formatting: Google Sheets / CSV.
* Version Control: Git/GitHub.
